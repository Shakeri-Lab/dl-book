\section{One Unit and One Layer}

For a column-vector input $x\in\mathbb{R}^{D_{\mathrm{in}}}$, one output unit computes
\[
  z_j=w_j^\top x+b_j.
\]
Stacking $D_{\mathrm{out}}$ row vectors gives
\[
  z=Wx+b,
  \qquad
  W\in\mathbb{R}^{D_{\mathrm{out}}\times D_{\mathrm{in}}}.
\]
This map is affine when $b\ne0$. For examples stored as rows,
\[
  Z=XW^\top+b.
\]
A nonlinear layer then applies an activation elementwise or according to another
declared axis contract.
